### 一个方便查找杀软进程的脚本

最近在学习内网渗透的时候发现一个个对比进程太麻烦了，发现了这位老哥给的一个[字典](https://raw.githubusercontent.com/r00tSe7en/get_AV/master/av.json)，自己改一下，方便自己使用

### Usage：

三种模式：

1,2都是检查对应的进程，1比2要慢因为需要从那位老哥的github上拖字典，2是直接使用的本地json文件   

 3是更新本地字典
